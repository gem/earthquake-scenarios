# RECORDING STATIONS

![](recording_stations.png)

A unified database of recording stations is available in `Stations_Unique.csv` file.
When the same station data is found in multiple sources, the information from the highest priority source is preferred to avoid duplication.

The reference datasets were used in the following priority order:
- `Stations_INGV`: INGV Recording stations. Raw file: `INGV_records.zip` Obtained from https://itaca.mi.ingv.it/ItacaNet_32/#/event/IT-1997-0137. Last accessed December 2022.
- `Stations_USGS`: USGS Recording stations. Raw file [`stationlist.json`]https://earthquake.usgs.gov/earthquakes/eventpage/usp0008916/shakemap/stations. Last accessed December 2022.
